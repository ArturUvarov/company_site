Artur Uvarov

Mokykloje pradėjau mokytis C++ ir Java, bet šios kalbos man pasirodė per sudėtingos, todėl kurį laiką praradau susidomėjimą programavimu. Vėliau atradau JavaScript – ji buvo aiškesnė, lankstesnė ir įdomesnė, todėl pradėjau ją mokytis.

Ignas Kasparaitis

Full-stack programuotojas. Moku programuoti JavaScript,projektai TypeScript, Lua ir Python kalbomis, taip pat gerai išmanau įvairius įrankius, tokius kaip Git, Docker ir kt. Pastaruosius kelerius metus kūriau asmeninius projektus, o neseniai nusprendžiau žengti žingsnį į profesionalų pasaulį – todėl esu čia. Man patinka matyti užbaigtą darbą, todėl greitai įgyvendintas projektas su gerai apgalvotais sprendimais yra tai, kas mane motyvuoja.

Daniil Gnatiuk

Esu JavaScript programuotojas, turintis patirties tiek su „frontend", tiek su „backend" technologijomis. Kuriu modernias, patogias naudoti internetines aplikacijas, naudodamas React ir Tailwind CSS vartotojo sąsajai kurti. „Backend" pusėje dirbu su Node.js ir Express.js, naudoju SQL duomenų bazes bei įgyvendinu autentifikaciją naudojant JWT. Taip pat moku dirbti su Drizzle ORM, valdau projektus pagal gerąsias programavimo praktikas ir orientuojuosi į efektyvų, saugų bei optimizuotą kodą.
